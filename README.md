## 📖 블로그 소개 및 이용 방법

### 🌌 블로그 소개
**MathAstro**는 수학(Math)과 천문/물리(Astro)를 중점으로 다양한 주제들을 다루는 Quarto 기반의 블로그이며, 두 명의 운영진이 공동으로 운영하고 있습니다.

### 🛠️ 이용 방법
각 학문 분야별로 세부 주제들이 분류되어 있습니다. 예를 들어 수학에서 추상대수학(Abstract Algebra)의 경우 Group Theory, Field Theory, Ring Theory 등으로 나뉘어 있습니다.
